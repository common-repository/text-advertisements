== Description ==
After Activation the plugin use [Text-Advertisements] shortcode and another one written in shorcodes text file.

Features:

* Easily you can manage your advertisements

Language Support:

* English
* Persian (فارسی)

== Installation ==
1. Upload `Text-Advertisements` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage your ads in 'text advertisements' menu
4. use shortcodes in shortcodes text file
